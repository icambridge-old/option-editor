== Description ==

Allows users to edit the internal WordPress options without digging about the database. If you don't know what that means or think it's a better interface for configuring your WordPress install this isn't for you. You will proably end up breaking your WordPress install. This plugin has a very limited use.

*Requires PHP 5.2 or above*

== Installation ==

1. Upload `option-editor` folder to `/wp-content/plugins/` directory
2. Go to Plugins and activate `Option Editor` 
3. Go to `Option Editor` in the `Tools` menu.
